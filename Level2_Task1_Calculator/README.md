# HoloCalc — Quantum Interface UI

A premium, modern sci-fi holographic web-based calculator featuring crisp fluid typography, deep space glow layers, and complex mathematical operation chaining.

## Features
- **Chained Logic Matrix:** Computes continuous inputs sequentially adhering to standard structural evaluations.
- **Micro-Animations:** Fluid mouse and layout response scaling down elements to `0.93` upon active interaction.
- **Adaptive Screen Scalar:** Automatic dynamic viewport calculation system shrinks font vectors asynchronously on edge length overrides to maintain pristine layouts up to 16 digits.
- **Physical Keyboard Injection mapping:** Full physical deck typing integration.
