# Space Thermometer — Temperature Converter

A premium single-page Temperature Converter web app built using pure HTML, CSS, and JavaScript.

Designed with a modern space-inspired UI, the application converts temperatures instantly between Celsius, Fahrenheit, and Kelvin while visually representing temperature changes using a glowing animated orb.

---

## Features

- Real-time temperature conversion
- Convert between:
  - Celsius
  - Fahrenheit
  - Kelvin
- Smooth glowing orb animation based on temperature
- Dynamic color transitions:
  - Blue → Cold temperatures
  - Golden → Moderate temperatures
  - Orange/Red → Hot temperatures
- Input validation for numeric values
- Fully responsive design
- Minimal and modern interface
- Pure frontend project (No frameworks or libraries)

---

## Design Theme

The interface follows a dark space-style visual theme with:

- Deep dark background
- Golden accent highlights
- Soft glassmorphism effects
- Animated ambient stars
- Space Grotesk typography

The goal was to create a clean and premium experience instead of a traditional utility-style converter.

1. Download the project file
2. Open the `.html` file in any browser

---

## Temperature Logic

The converter automatically updates all values in real time based on the selected input unit.

### Orb Color Behavior

| Temperature (°C) | Orb Color |
|------------------|-----------|
| Below 0°C | Blue |
| 0°C – 30°C | Golden |
| Above 30°C | Orange/Red |
