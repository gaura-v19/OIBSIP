# To-Do Web App — Level 2 Task 3

## Overview
A task management web app built with vanilla HTML, CSS and 
JavaScript. No frameworks, no dependencies, no build tools.

## Features
- Add, edit and delete tasks instantly
- Priority levels — High, Medium, Low
- Due dates with automatic overdue detection
- Filter tasks — All, Active, Completed views
- Live progress bar with completion percentage
- Drag and drop to reorder tasks
- Data saved to localStorage — survives page refresh
- Toast notifications for all actions
- Responsive across all screen sizes

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
