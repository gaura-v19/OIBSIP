## Overview

A task management web app built with vanilla HTML, CSS and JavaScript. No frameworks, no dependencies, no build tools.

---

## Features

| Feature | Description |
|---|---|
| Task Management | Add, edit and delete tasks instantly |
| Priority Levels | High, Medium and Low with color coding |
| Due Dates | Set deadlines with automatic overdue detection |
| Filters | Switch between All, Active and Completed views |
| Progress Tracking | Live progress bar with completion percentage |
| Drag & Drop | Reorder tasks by dragging them |
| Persistence | Data saved to localStorage — survives page refresh |
| Notifications | Toast messages for every action |
| Responsive | Works across all screen sizes |
