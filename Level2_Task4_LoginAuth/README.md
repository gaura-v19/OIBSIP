# AuthKit — Animated Mascot Login System

## Overview
A premium login and register authentication web app featuring a fully 
CSS-animated mascot that reacts to your cursor and actions in real time. 
No images, no SVG, no canvas — pure CSS and JavaScript only.

## Features
- Pure CSS mascot — zero images, zero SVG, zero canvas
- Cursor tracking — head and pupils follow your mouse in real time
- Password field — mascot covers eyes with hands automatically
- Error state — mascot shakes head on failed validation
- Success state — mascot jumps and smiles on successful login
- Natural blinking every 2-4 seconds
- Register, Login and Welcome page flows
- localStorage — users persist across sessions
- Full validation — empty fields, email format, password length, duplicate email

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run
1. Clone the repo
2. Open index.html in any modern browser
3. No build step or installation needed
